## Decisões técnicas

* -Decidi utilizar o SSE pela liberdade, agilidade e compatibilidade com a estrutura do projeto.
* -Apliquei o Express para Facilitar a criação do servidor com Node e maximizar a comunicação com diferentes URLs.
* -Apliquei o Cors para Gerenciar a comunicação entre as portas, React e Node.
* -Criei o projeto pelo NPX como descrito na documentação do React.   
* -Poderia ter Utilizado o (vite) para criar um projeto mais limpo e escalavel, mas decidi seguir com
o npx por está mais acostumado com a criação padrão dos projetos em React.
* -Primeiro estruturei todas as pastas e files necessarias para iniciar o projeto.
* -Reutilizei um box de Login ja criado por mim em outro projeto para adiantar o processo de desenvolvimento.
* -Desenvolvi a base dos componentes da tela para otimizar as etapas seguintes do projeto.
* -Inicializei a configuração e primeiros testes do servidor backend.
